Good Morning My Dear Pondati - Theme + Photo Version

How to use:
1. Extract/unzip this folder.
2. The uploaded couple photo is already included as wife-photo.jpg.
3. Upload these files directly to your GitHub repository root:
   - index.html
   - style.css
   - script.js
   - wife-photo.jpg
4. Commit changes.
5. Wait for GitHub Pages to update.

New in this version:
- Dark mode and light mode toggle.
- Your uploaded couple photo is already added.
- Photo alignment is adjusted for mobile.
- No overlapping text.
- Clean final message box at the bottom.
